<p align="center">
A MO2 theme based on the One Dark theme for the Atom text editor.
</p>

Uses some flash assets from  
[Paper Dark](https://www.nexusmods.com/skyrim/mods/64439).  
Big thanks to **AlcoholSwabs**!

---

## Installation

Copy and paste the `stylesheets` folder into your Mod Organizer directory.  
In **MO Settings → General → Style**, select **ATOM.qss**.

Under **MO Settings → General → Colors**, it is best to set:
- **Mod Contains selected Plugin** → `255, 255, 255`
- **Plugin is Contained in selected Mod** → `255, 255, 255`
